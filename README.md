# Тестовое задание 

Общие правила: 

1.Верстка с соблюдением методологии БЭМ
2.Семантически правильная верстка.
  ошибки - плохо
  предупреждения - допускаются
3.Кросбраузерная верстка
  css дожен поддерживаться всеми современными браузерами

Структура блока:

1.Основной текст
  сплошной абзац текста, без разбивки на строки
  верстка должна предусматривать возможность добавления изображений между любыми словами
  изображения могут добавляться, удаляться или перемещаться в любую часть текста, верстка должна автоматически подстраиваться под такие изменения и не ломаться
  изображения могут быть разной ширины
2.Кнопка
  в html должен быть чисный тег button без лишней графики или svg
  стилизация, только возможностями css
  фоновую линию тоже пишем на css
  геометрия кнопки не должна ломаться при увеличении или уменьшении объема текста, при этом ширина должна подстраиваться
